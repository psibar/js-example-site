# js-example-site

A showcase of dynamic and interactive web features built with vanilla JavaScript, HTML, and CSS.

## Live Demo

Visit the live site at: [https://psibar.github.io/js-example-site](https://psibar.github.io/js-example-site)

## Features

This example site demonstrates:

- **Interactive Counter** - Increment, decrement, and reset functionality with smooth animations
- **Dynamic List** - Add and remove items dynamically with fade animations
- **Color Theme Switcher** - Multiple theme options with localStorage persistence
- **Interactive Animations** - CSS animations triggered by JavaScript events
- **Data Visualization** - Interactive Plotly charts with multiple data curves

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- Vanilla JavaScript (ES6+)
- localStorage API
- Plotly.js for data visualization
